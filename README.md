# Karo Pitch – Startup Pitch Platform

This project is a demo landing page created for the **KaroStartup Internship Assignment**.

Karo Pitch is a platform designed to help startup founders from across India pitch their businesses to investors and raise funding.

The goal of this demo website is to present the concept of the platform through a clean and modern startup-style landing page.

---

## Features

- Modern startup landing page design
- Hero section with strong call-to-action
- About Karo Pitch section
- How It Works process
- Startup categories that can apply
- Investor introduction section
- Featured startup cards
- About KaroStartup section
- Final call-to-action for founders

---

## Technologies Used

- HTML
- CSS
- JavaScript
- AI-assisted design tools

This project was created as part of an internship assignment to demonstrate:
